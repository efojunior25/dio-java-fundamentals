# Perguntas e Respostas — Instalação do JDK no Linux

## Conceituais

1. O que é uma distribuição do JDK?
   Uma implementação do Java baseada no OpenJDK.

2. O que é o Amazon Corretto?
   A distribuição de Java mantida pela AWS.

3. Por que o professor escolheu o Corretto?
   Porque é gratuito e tem suporte de uma grande empresa.

4. Oracle JDK é gratuito?
   Apenas para desenvolvimento.

5. Para produção o Oracle JDK é pago?
   Sim.

6. Todas as distribuições são compatíveis?
   Sim.

7. O que é OpenJDK?
   O código base open source do Java.

8. O que é o JDK?
   O kit de desenvolvimento do Java.

9. Java depende da Oracle?
   Não.

10. Java é uma plataforma?
    Sim.
