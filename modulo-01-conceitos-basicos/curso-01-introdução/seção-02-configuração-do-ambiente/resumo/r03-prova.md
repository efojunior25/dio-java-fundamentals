# 3. Prova — 10 Questões

### 1. O que é o JDK?
**Resposta:** O kit de desenvolvimento do Java.

### 2. Qual variável aponta para o JDK?  
**Resposta:** JAVA_HOME.

### 3. Qual pasta deve ser adicionada ao PATH?  
**Resposta:** %JAVA_HOME%\bin.

### 4. Qual comando mostra a versão do Java?  
**Resposta:** java -version.

### 5. O que é uma distribuição do JDK?  
**Resposta:** Uma implementação do Java baseada no OpenJDK.

### 6. O Amazon Corretto é pago?  
**Resposta:** Não.

### 7. O que é o SDKMAN?  
**Resposta:** Um gerenciador de versões de SDKs.

### 8. Qual comando lista versões de Java no SDKMAN?  
**Resposta:** sdk list java.

### 9. O SDKMAN funciona no Windows?  
####   **Resposta:** Sim, via WSL.

### 10. Qual versão de Java o curso usa?
**Resposta:** Java 21.
