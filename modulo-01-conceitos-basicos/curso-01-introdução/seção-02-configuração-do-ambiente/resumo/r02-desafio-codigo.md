# Desafio — Verificador de Ambiente Java

Crie um programa Java que:

1. Mostre a versão do Java
2. Mostre o fornecedor do JDK
3. Mostre o valor da variável JAVA_HOME
4. Informe se a versão é Java 21

Saída esperada:

```
Java Version: 21.0.1
Vendor: Amazon Corretto
JAVA_HOME: /usr/lib/jvm/java-21-amazon-corretto
Status: Versão correta para o curso
```



Dica: utilize:
- `System.getProperty()`
- `System.getenv()`
