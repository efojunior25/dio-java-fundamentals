# Perguntas de Entrevista (5)

### 1. Qual a diferença entre JDK e JVM?  
   A JVM executa o código e o JDK inclui a JVM e as ferramentas de desenvolvimento.

### 2. Para que serve o JAVA_HOME?  
   Para indicar onde o JDK está instalado.

### 3. Por que usar SDKMAN?  
   Para gerenciar várias versões do Java facilmente.

### 4. Por que usar Amazon Corretto?  
   Porque é gratuito e mantido pela AWS.

### 5. Como saber se o Java está corretamente instalado?  
   Executando `java -version` e `javac -version`.
