## Lições técnicas
- Sem JDK não existe Java
- PATH é obrigatório
- OpenJDK é suficiente para quase tudo

## Lições de carreira
- Configurar ambiente é parte do trabalho
- Bons desenvolvedores dominam suas ferramentas

## Glossário
- JDK: Java Development Kit
- JVM: Java Virtual Machine
- PATH: Variável de ambiente

## Checklist
- [ ] Instalei o JDK
- [ ] Configurei o PATH
- [ ] Testei java -version
- [ ] Testei javac -version

## Comparações
- Oracle JDK x OpenJDK
- Instalação manual x SDKMAN

## Erros comuns
- Esquecer de configurar o PATH
- Instalar JRE em vez de JDK
