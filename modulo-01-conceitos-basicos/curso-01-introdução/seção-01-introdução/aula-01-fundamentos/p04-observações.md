## Lições técnicas
- Java é mais que uma linguagem: é um ecossistema (JVM, frameworks, libs)
- Annotations podem gerar código automaticamente
- Dependências são fundamentais em projetos reais

## Lições de carreira
- Começar com tecnologia legada não impede crescimento
- Persistência e base sólida fazem diferença
- Migrar de stack é possível com fundamentos

## Glossário
- JVM: Java Virtual Machine
- JPA: Java Persistence API
- Annotation Processor: Ferramenta que gera código em tempo de compilação
