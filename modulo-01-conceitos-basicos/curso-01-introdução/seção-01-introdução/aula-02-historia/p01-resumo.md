# Vídeo 02 — História do Java

## 1. O surgimento do Java
- Em 1991, na **Sun Microsystems**, foi criado o **Green Team**
- O time foi liderado por **James Gosling**
- O objetivo era criar uma tecnologia para:
    - Casas inteligentes
    - Dispositivos conectados
    - Automação residencial
- O desafio era:
    - Sistemas operacionais diferentes
    - Falta de suporte a multiplataforma

## 2. O primeiro grande problema
Na época:
- Windows, Linux e Mac usavam comandos diferentes
- Um programa feito para um SO não funcionava em outro
- Era necessário reescrever código

### Solução criada
Criar uma linguagem:
- Independente de sistema operacional
- Multiplataforma

Assim surgiu o **GreenTalk**

## 3. GreenTalk → Oak → Java
- GreenTalk foi a primeira linguagem
- Depois foi renomeada para **Oak**
- Oak era uma referência a uma árvore que James Gosling via da janela

## 4. O projeto Star7
- Um controle remoto para casas inteligentes
- Usava Oak
- Introduziu o mascote **Duke** (o pato do Java)
- O mercado não adotou o produto

## 5. A virada com a internet
- Em 1994–1995 surgiu o boom da internet
- Navegadores populares:
    - Mosaic
- A Sun criou:
    - O navegador **WebRunner** (depois HotJava)

## 6. O grande diferencial
O navegador HotJava permitia:
- Rodar programas dentro da página
- Esses programas eram chamados de **Applets**
- Isso era revolucionário na época

## 7. O nome Java
- Oak não podia ser registrado
- A equipe escolheu o nome **Java**
- Inspirado no café forte chamado **Java Coffee**
- Navegador: **HotJava**
- Linguagem: **Java**

## 8. O slogan
**Write Once, Run Anywhere**
> Escreva uma vez, execute em qualquer lugar

Isso era possível porque:
- O código Java roda na **JVM**
- A JVM existe para vários sistemas operacionais

## 9. Expansão do Java
- 1996: Lançamento do **JDK**
- Criação de várias plataformas:
    - Java SE
    - Java Card
    - Java Mobile (antes do Android)

## 10. Problema de evolução
- Java demorava para receber novidades
- Exemplo:
    - Expressões lambda só chegaram no Java 8 (2014)
- Motivo:
    - Forte compromisso com compatibilidade
    - Estabilidade
    - Segurança

## 11. Oracle e o modelo LTS
- A Sun foi adquirida pela **Oracle**
- Novo modelo de versões:
    - LTS (Long Term Support)
    - Versões intermediárias para testes

Exemplo:
- Java 8 (LTS)
- Java 11 (LTS)
- Java 17 (LTS)
- Java 21 (LTS) → versão usada no curso

## 12. Mitos sobre Java

### "Java é lento"
- Surgiu por causa de problemas no Java 5
- Hoje Java pode ter desempenho próximo a C/C++

### "Java é verboso"
- Verdade em parte
- Mas as versões modernas estão reduzindo isso

## 13. Java no mercado
Empresas que usam Java:
- Netflix
- PayPal
- Slack
- PicPay
- LinkedIn
- Airbnb
- Microsoft

## 14. Java como ferramenta
- Nenhuma linguagem é perfeita para tudo
- Cada linguagem tem seu contexto
- Exemplo:
    - Twitter migrou de Ruby para Java por desempenho
    - PicPay usa PHP, Java e Kotlin conforme o cenário

## 15. O futuro do Java
- Java continua evoluindo
- É amplamente usado em empresas
- Mesmo tecnologias legadas continuam úteis
- Java dificilmente desaparecerá
