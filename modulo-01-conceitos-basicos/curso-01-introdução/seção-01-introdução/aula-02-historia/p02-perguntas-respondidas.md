# Perguntas e Respostas — História do Java

## Conceituais

1. Quem criou o Java?
   James Gosling.

2. Em que empresa o Java nasceu?
   Sun Microsystems.

3. Em que ano o Java começou a ser desenvolvido?
1991.

4. Qual era o objetivo original?
   Conectar dispositivos em casas inteligentes.

5. O que era o Green Team?
   O time que criou o Java.

6. O que era o GreenTalk?
   Primeira versão da linguagem Java.

7. O que foi Oak?
   Nome intermediário da linguagem.

8. Por que o nome Oak foi abandonado?
   Não pôde ser registrado.

9. De onde veio o nome Java?
   Do café Java Coffee.

10. Qual é o slogan do Java?
    Write Once, Run Anywhere.

## Técnicas

11. O que é JVM?
    Máquina virtual que executa programas Java.

12. O Java é multiplataforma?
    Sim.

13. O que eram applets?
    Programas Java que rodavam no navegador.

14. Qual navegador usava Java?
    HotJava.

15. O que é o JDK?
    Kit de desenvolvimento Java.

16. O que é uma versão LTS?
    Versão com suporte prolongado.

17. Quais são versões LTS importantes?
    8, 11, 17 e 21.

18. Por que o Java demorava a evoluir?
    Por causa da compatibilidade.

19. O Java 8 trouxe o quê?
    Expressões lambda.

20. Quem é a empresa que hoje mantém o Java?
    Oracle.

## Mercado

21. Java ainda é usado?
    Sim, amplamente.

22. Cite uma empresa que usa Java.
    Netflix.

23. O Java é lento?
    Não nas versões modernas.

24. O Java é verboso?
    Sim, mas vem melhorando.

25. Java pode competir com C++?
    Sim em muitos cenários.

## Estilo entrevista júnior

26. Por que Java é multiplataforma?
    Porque roda na JVM.

27. O que é LTS?
    Long Term Support.

28. Por que empresas usam Java?
    Estabilidade e desempenho.

29. Java vai morrer?
    Muito improvável.

30. Posso usar Java em backend?
    Sim, é um dos principais usos.

31. Java é bom para sistemas grandes?
    Sim.

32. Java é usado em fintechs?
    Sim, como o PicPay.

33. O Java roda em celular?
    Sim, já rodou no Java Mobile.

34. O Android veio do Java?
    Sim, o ecossistema é baseado em Java.

35. O Java suporta múltiplas linguagens?
    Sim, pela JVM.

36. Kotlin roda onde?
    Na JVM.

37. Scala roda onde?
    Na JVM.

38. Ruby roda na JVM?
    Não nativamente.

39. Java é uma plataforma ou só linguagem?
    Uma plataforma.

40. O que garante a compatibilidade?
    A JVM e as especificações.
