## Lições técnicas
- Java é uma plataforma, não só uma linguagem
- A JVM permite multiplataforma
- LTS garante estabilidade

## Lições de carreira
- Nenhuma linguagem serve para tudo
- Bons engenheiros escolhem a ferramenta certa
- Migrar de stack é normal

## Glossário
- JVM: Java Virtual Machine
- JDK: Java Development Kit
- LTS: Long Term Support
- Applet: Programa Java no navegador
