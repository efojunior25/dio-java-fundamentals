# Instalação do IntelliJ IDEA

## Contexto da aula
A aula apresenta o processo de instalação do IntelliJ IDEA, IDE usada para desenvolvimento em Java. O foco é garantir que o ambiente esteja pronto para criar, compilar e executar projetos Java.

---

## O que é o IntelliJ IDEA
O IntelliJ IDEA é uma IDE (Ambiente de Desenvolvimento Integrado) que oferece:
- Editor de código inteligente
- Compilação e execução de projetos
- Integração com Maven e Gradle
- Ferramentas de depuração (debug)
- Suporte nativo a Java

Existem duas edições principais:
- Community (gratuita)
- Ultimate (paga, com mais recursos)

---

## Download
A instalação começa no site oficial da JetBrains.  
O usuário escolhe a edição desejada (normalmente a Community) e baixa o instalador para o Windows.

---

## Processo de instalação
Após executar o instalador:
- O assistente de instalação é iniciado
- O usuário escolhe o diretório de instalação
- Pode selecionar opções como:
    - Atalho na área de trabalho
    - Associação de arquivos `.java`
    - Inclusão no PATH

O processo copia os arquivos da IDE para o computador.

---

## Primeira execução
Ao abrir o IntelliJ pela primeira vez:
- A IDE pede algumas configurações iniciais
- Pode solicitar importação de configurações anteriores
- O usuário escolhe o tema visual
- Plugins padrão são configurados

---

## Finalidade da IDE
O IntelliJ será usado nas próximas aulas para:
- Criar projetos Java
- Escrever código
- Executar programas
- Trabalhar com Maven e Gradle
