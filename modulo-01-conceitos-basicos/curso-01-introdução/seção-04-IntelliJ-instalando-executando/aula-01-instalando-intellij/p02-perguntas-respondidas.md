# Perguntas e Respostas — Instalando o IntelliJ

1. O que é o IntelliJ IDEA?
   Uma IDE para desenvolvimento Java.

2. Quem desenvolve o IntelliJ?
   A JetBrains.

3. Qual edição é gratuita?
   Community.

4. Qual edição é paga?
   Ultimate.

5. Para que serve uma IDE?
   Para escrever, compilar e executar código.

6. O IntelliJ suporta Java?
   Sim.

7. O IntelliJ substitui o Java?
   Não, ele usa o Java.

8. Onde se baixa o IntelliJ?
   No site da JetBrains.

9. O que é um instalador?
   Um programa que copia a IDE para o sistema.

10. Posso escolher o diretório de instalação?
    Sim.

11. O IntelliJ pode criar atalhos?
    Sim.

12. Ele pode associar arquivos .java?
    Sim.

13. O IntelliJ integra Maven?
    Sim.

14. O IntelliJ integra Gradle?
    Sim.

15. A IDE compila código?
    Sim.

16. A IDE executa programas?
    Sim.

17. O IntelliJ tem debug?
    Sim.

18. O IntelliJ funciona no Windows?
    Sim.

19. Ele funciona no Linux?
    Sim.

20. Ele funciona no macOS?
    Sim.

21. A IDE precisa do Java?
    Sim.

22. O IntelliJ é uma JVM?
    Não.

23. O IntelliJ cria projetos?
    Sim.

24. O IntelliJ gera .class?
    Sim, ao compilar.

25. O IntelliJ substitui o terminal?
    Não, ele apenas facilita.

26. Plugins podem ser instalados?
    Sim.

27. Tema visual pode ser escolhido?
    Sim.

28. O IntelliJ importa configs antigas?
    Sim.

29. O IntelliJ é usado no mercado?
    Sim.

30. IDEs aumentam produtividade?
    Sim.

31. O IntelliJ é open source?
    A Community sim.

32. O IntelliJ executa código Java?
    Sim.

33. O IntelliJ é obrigatório?
    Não, mas recomendado.

34. O IntelliJ pode usar Maven?
    Sim.

35. O IntelliJ pode usar Gradle?
    Sim.
