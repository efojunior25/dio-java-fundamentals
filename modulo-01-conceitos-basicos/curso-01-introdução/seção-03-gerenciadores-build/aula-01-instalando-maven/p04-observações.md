## Lições técnicas
- Maven precisa de JAVA_HOME
- MAVEN_HOME aponta para o Maven
- PATH precisa incluir bin
- Permissões do Windows podem quebrar tudo

## Lições de carreira
- Problemas de ambiente são comuns
- Saber diagnosticar PATH e permissões é essencial
