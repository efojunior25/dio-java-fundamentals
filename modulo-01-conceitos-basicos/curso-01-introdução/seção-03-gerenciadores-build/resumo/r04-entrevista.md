# Perguntas de Entrevista — Gerenciadores de Build

1. Qual a diferença entre Maven e Gradle?
   O Maven usa XML e é mais padronizado; o Gradle usa scripts e é mais flexível.

2. Por que usamos gerenciadores de build?
   Para automatizar compilação, dependências, testes e empacotamento.

3. O que é MAVEN_HOME?
   A variável que aponta para a instalação do Maven.

4. O que acontece se o PATH estiver errado?
   Os comandos mvn e gradle não funcionam.

5. Por que um desenvolvedor precisa dominar essas ferramentas?
   Porque todo projeto Java profissional depende de um sistema de build confiável.
