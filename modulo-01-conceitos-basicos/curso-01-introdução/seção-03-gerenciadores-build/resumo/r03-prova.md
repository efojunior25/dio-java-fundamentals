# Prova — Gerenciadores de Build

1. O que é build?
   R: Processo de transformar código-fonte em software executável.

2. O que um gerenciador de build faz?
   R: Automatiza o processo de build.

3. Quais são os dois principais gerenciadores Java?
   R: Maven e Gradle.

4. Qual variável aponta para o Maven?
   R: MAVEN_HOME.

5. Qual variável aponta para o Gradle?
   R: GRADLE_HOME.

6. O que deve ser adicionado ao PATH?
   R: A pasta bin do Maven ou Gradle.

7. Qual comando testa o Maven?
   R: mvn -version.

8. Qual comando testa o Gradle?
   R: gradle -v.

9. O Maven usa XML ou scripts?
   R: XML.

10. O Gradle usa XML ou scripts?
    R: Scripts.
